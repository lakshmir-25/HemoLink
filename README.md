# HemoLink

HemoLink is a simple blood donor finding website built for a hackathon.

## Features
- User login with phone number & password
- Register as a donor with:
  - Name
  - Blood group
  - Phone number
  - City
  - Gender
  - Age
  - Availability toggle
  - Consent for sharing contact info
- Search donors by blood group and city
- Notifications sent to available donors (simulated via console)
  
## How to Run
1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Login using any phone number and password.
4. Navigate to "Register as Donor" to register.
5. Navigate to "Search Donor" to find available donors.

## Tech Stack
- HTML
- CSS
- JavaScript
- node + express
